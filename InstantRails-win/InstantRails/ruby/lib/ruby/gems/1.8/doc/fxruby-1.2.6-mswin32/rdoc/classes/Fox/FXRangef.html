center()

Return the center of the box (a point, as an FXVec3f instance).

contains?(x, y, z)

Returns true if this range contains the point (x, y, z).

contains?(p)

Returns true if this range contains the point p (an FXVec3f instance)

contains?(aRange)

Returns true if this box properly contains aRange (another FXRangef instance)

contains?(aSphere)

Returns true if this box properly contains aSphere (an FXSpheref instance)

corner(c)

Return the _c_th corner of this box (an FXVec3f instance). Raises IndexError if c is less than zero or greater than 7.

depth()

Return the depth of the box

diagonal()

Return the diagonal vector for the box (an FXVec3f instance).

diameter()

Return the length of the diagonal of the box

empty?()

Returns true if this is an empty range (i.e. if any of the side lengths is zero or less).

height()

Return the height of the box

include!(*args)

Include the given range or point into this range and return a reference to self. Valid forms are: 

    range.include!(aRange)  -> range
    range.include!(x, y, z) -> range
    range.include!(vec)     -> range
    range.include!(sphere)  -> range
intersect(plane)

Intersect box with a plane ax+by+cz+w; returns -1, 0 or 1.

intersection(other)

Return a new FXRangef instance which is the intersection of this box and another box.

intersects?(u, v)

Return true if the ray from u to v (both FXVec3f instances representing the ray endpoints) intersects this box.

longest()

Return the length of the longest side of the box

overlaps?(other)

Returns true if any part of this range overlaps the other range.

shortest()

Return the length of the shortest side of the box

union(other)

Return a new FXRangef instance which is the union of this box and another box.

width()
