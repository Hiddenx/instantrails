Class Fox::FXIconDict
In: lib/fox12/aliases.rb
rdoc-sources/FXFileDict.rb
Parent: Object

An FXIconDict instance stores a mapping between the file names for icon files and FXIcon instances built from those icon files. Unlike a regular "dictionary" or hash-like object, the FXIconDict constructs and owns its own data (the FXIcon instances).

If no icon path is specified at construction time, a default icon path of "~/.foxicons:/usr/local/share/icons:/usr/share/icons" is used.

The particular subclass of FXIcon used for a given icon file is deduced from the file extension.
