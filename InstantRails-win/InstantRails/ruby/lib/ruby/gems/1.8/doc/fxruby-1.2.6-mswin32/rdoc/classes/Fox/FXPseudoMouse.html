Class Fox::FXPseudoMouse
In: lib/fox12/pseudomouse.rb
Parent: FXObject

Based on a suggestion from Hugh Sasse on the fxruby-users mailing list, an FXPseudoMouse object provides a simple means to operate widgets programmatically, to aid test driven design. An FXPseudoMouse instance can be pointed at an FXObject and will manage the sending of events to it.
