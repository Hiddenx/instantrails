Class Camping::Controllers::NotFound < Object

(in files lib/camping-unabridged.rb )

The NotFound class is a special controller class for handling 404 errors, in case you’d like to alter the appearance of the 404. The path is passed in as p. 

  module Camping::Controllers
    class NotFound
      def get(p)
        @status = 404
        div do
          h1 'Camping Problem!'
          h2 "#{p} not found"
        end
      end
    end
  end
