Public Instance methods

(-();)

Returns a new FXVec instance which is the negation of this one.

*(n)

Returns a new FXVec instance obtained by memberwise multiplication of this vector’s elements by the scalar n.

+(other)

Returns a new FXVec instance obtained by memberwise addition of the other FXVec instance with this one.

-(other)

Returns a new FXVec instance obtained by memberwise subtraction of the other FXVec instance from this one.

/(n)

Returns a new FXVec instance obtained by memberwise division of this vector’s elements by the scalar n. Raises ZeroDivisionError if n is identically zero.

==(other)

Return true if this vector is equal to other.

[](index)

Returns the element at index, where index is 0, 1 or 2. Raises IndexError if index is out of range.

[]=(index, value)

Set the element at index to value and return value. Raises IndexError if index is out of range.

cross(other)

Return the cross product of this vector and other.

dot(other)

Returns the dot (scalar) product of this vector and other.

hi(other)

Return a new FXVec instance which is the greater of this vector and other.

len()

Return the length (magnitude) of this vector.

lo(other)

Return a new FXVec instance which is the lesser of this vector and other.

normalize()

Return a new FXVec instance which is a normalized version of this one.

normalize!()

Normalize this vector and return a reference to it.

to_a()

Return a new Array instance with this vector’s elements as its members.
